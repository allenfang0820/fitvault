## 4. 当前 UI 遗漏内容

- 当前 UI 尚未完整形成“身体状态如何变化 / 为什么失衡 / 在哪里开始崩 / 什么因素导致崩溃”的分析叙事。
- 当前主图是心率、速度、GAP 叠加图，不是设计稿的多维分层时间轴。
- Layer 1 疲劳带、Layer 2 事件标记、Layer 3 派生指标曲线尚未形成清晰主视觉层级。
- 缺少独立状态阶段概览横条。
- 派生指标没有回正为设计稿中的紧凑横向指标条和小趋势视觉。
- 右侧关键摘要、崩溃触发因素、基准核心因素、生理冲击点尚未完整覆盖。
- 底部事件时间线、负荷与能量分析、状态解释 / 建议、对比分析尚未完整覆盖。
- 配色、信息密度和深色玻璃运动仪表盘质感仍与设计稿存在差距。
